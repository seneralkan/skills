# Review Commands

Run these commands before final recommendations.

```bash
go fmt ./...
go vet ./...
go test ./...
go test -race ./...
staticcheck ./...
golangci-lint run
govulncheck ./...
```

## Optional deep checks

```bash
go test -run TestName -v ./...
go test -bench . -benchmem ./...
go tool pprof -http=:8080 cpu.pprof
```
